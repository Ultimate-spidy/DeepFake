# Source Code

This directory contains the main source code for the DeepFake detection system.

## Structure
- `models/` - Model architecture definitions
- `data/` - Data loading and preprocessing utilities  
- `training/` - Training scripts and loops
- `inference/` - Inference and prediction scripts
- `utils/` - Utility functions and helpers

## Getting Started
1. Add your Python modules to the appropriate subdirectories
2. Include `__init__.py` files to make directories importable
3. Follow PEP 8 style guidelines
4. Add docstrings and type hints to your functions
