# Quick Setup Script

This script helps you quickly set up your local project in the DeepFake repository.

## Usage

```bash
# Make the script executable
chmod +x setup.sh

# Run the setup
./setup.sh
```

The script will:
1. Check if required directories exist
2. Create missing directories
3. Set up basic Python package structure
4. Install dependencies (if requirements.txt exists)

If you prefer to set up manually, see [PROJECT_SETUP.md](../PROJECT_SETUP.md) for detailed instructions.
